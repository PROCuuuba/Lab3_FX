**Lab3_FX: Слайд-шоу с навигацией и фильтрацией**

Данный проект представляет собой третью лабораторную работу, посвященную разработке интерактивного приложения для просмотра изображений. Основное внимание уделено реализации паттерна проектирования Итератор и работе с медиа-контентом в JavaFX.

**Функциональные возможности**

Приложение реализует полноценный механизм просмотра графических файлов со следующими функциями:

	•	Навигация: переходы вперед, назад, а также быстрый переход к первому и последнему кадру коллекции
	•	Паттерн Итератор: использование кастомного итератора для циклического обхода изображений и обработки пустых папок
	•	Фильтрация: возможность отбора файлов по типу расширения с динамическим обновлением счетчика позиций
	•	Анимация переходов: реализация визуальных эффектов (например, исчезновение или сдвиг) при смене слайдов
	•	Информационная панель: отображение данных о текущем файле, включая чтение EXIF-данных (дата съемки, параметры камеры)
